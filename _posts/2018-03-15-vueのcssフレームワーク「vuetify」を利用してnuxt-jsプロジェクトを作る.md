今回は、[Vuetify](https://vuetifyjs.com/ja/) を利用して、Nuxt.jsのプロジェクトを作成していきます。

# vueコマンドを利用してNuxt.jsプロジェクト作成する

```bash

$ vue init vuetifyjs/nuxt my-vuetify
$ cd my-vuetify
$ npm install

```
